# Instagram Mini Clone

This is a simple Instagram mini clone project that I am building to practice
full-stack development using MERN stack.

Right now the project contains basic frontend pages and backend setup.
More features will be added step by step.

---

## What is done till now

Frontend:
- Login page
- Signup page
- Home feed screen
- Profile page
- Routing using React Router

Backend:
- Node.js and Express server setup
- Basic folder structure (routes, models, middleware)
- Environment file configuration

---

## Tech Stack Used

Frontend:
- React
- Vite
- JavaScript
- CSS

Backend:
- Node.js
- Express.js
- MongoDB (schema will be added later)

---

## Project Folder Structure

instagram-mini-clone
- frontend
- backend

---

## How to Run the Project

Frontend:
1. Go to frontend folder
2. Run `npm install`
3. Run `npm run dev`

Backend:
1. Go to backend folder
2. Run `npm install`
3. Run `npm start`
